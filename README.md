
# Фінальний проєкт курсу "Basic Algorithms and Data Structures"

## Опис проєкту
Цей проєкт вирішує задачі, пов'язані з основними алгоритмами та структурами даних, включаючи роботу з однозв'язними списками, фракталами, графами та методами оптимізації. Кожне завдання включає реалізацію алгоритму та візуалізацію результатів.

## Завдання проєкту
1. **Завдання 1: Структури даних. Сортування. Робота з однозв'язним списком**
   - Реалізація однозв'язного списку, реверсування, сортування вставками та злиттям.
   
2. **Завдання 2: Рекурсія. Створення фрактала "дерево Піфагора"**
   - Програма використовує рекурсію для створення фрактала і відображає його з використанням бібліотеки `turtle`.

3. **Завдання 3: Алгоритм Дейкстри та графи**
   - Алгоритм Дейкстри для пошуку найкоротших шляхів в графі з використанням бінарної купи для оптимізації.

4. **Завдання 4: Візуалізація бінарних дерев**
   - Створення та візуалізація бінарних дерев за допомогою бібліотеки `networkx`.

5. **Завдання 5: Обхід бінарного дерева**
   - Реалізація обхідних алгоритмів для бінарних дерев: в глибину (DFS) і в ширину (BFS), з візуалізацією кожного кроку.

6. **Завдання 6: Жадібні алгоритми та динамічне програмування**
   - Реалізація жадібного алгоритму та алгоритму динамічного програмування для вибору страв з найбільшою калорійністю в межах обмеженого бюджету.

7. **Завдання 7: Метод Монте-Карло**
   - Симуляція підкидування двох кубиків з підрахунком ймовірностей кожної суми та порівняння результатів з аналітичними розрахунками.

## Приклад роботи
### Завдання 1: Однозв'язний список
```
Меню:
1. Додати елемент в список
2. Видалити елемент за значенням
3. Реверсувати список
4. Сортувати список (вставками)
5. Сортувати список (злиттям)
6. Знайти середнє значення елементів
7. Об'єднати два списки
8. Вивести список
9. Генерація випадкових даних
0. Вийти
```

### Завдання 2: Дерево Піфагора
Програма виводить фрактал у вигляді дерева з гілками, що змінюються за розміром та кутом. Користувач може вказати рівень рекурсії для побудови дерева.

### Завдання 3: Алгоритм Дейкстри
```
Найкоротші шляхи від вершини 0:
Вершина 0: 0
Вершина 1: 5
Вершина 2: 8
...
```

### Завдання 4: Візуалізація бінарних дерев
Програма побудує та візуалізує бінарні дерева, дозволяючи переглядати їх у графічному вигляді.

### Завдання 5: Обхід бінарного дерева
Виконується обхід дерева в глибину (DFS) та в ширину (BFS), з відображенням кожного кроку у вигляді кольорових змін на вузлах.

### Завдання 6: Жадібні алгоритми та динамічне програмування
Використовуються два підходи для вибору оптимальних страв у межах обмеженого бюджету: жадібний алгоритм та динамічне програмування.

### Завдання 7: Метод Монте-Карло
Моделюються випадкові кидки двох кубиків, обчислюються ймовірності для кожної суми та порівнюються з аналітичними розрахунками.

## Технології
- Python 3.x
- Бібліотеки:
  - `matplotlib` — для побудови графіків та візуалізацій
  - `networkx` — для роботи з графами
  - `colorama` — для кольорового виведення в консоль
  - `turtle` — для малювання фракталів

## Установка та запуск

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/mxmz-code/goit-algo-fp.git
   cd goit-algo-fp
   ```

2. Встановіть залежності:
   ```bash
   pip install matplotlib networkx colorama
   ```

3. Запустіть необхідний скрипт:
   - Для запуску завдання 1: `python task1.py`
   - Для запуску завдання 2: `python task2.py`
   - І так далі для інших завдань.

## Особливості та обмеження
- Завдання 2 використовує бібліотеку `turtle`, що вимагає графічного середовища для коректної роботи.
- Завдання 6 і 7 демонструють різні алгоритми оптимізації, зокрема жадібний та динамічне програмування для вибору страв.
- Програма працює в консольному режимі та надає інтерфейс для взаємодії з користувачем через меню.
